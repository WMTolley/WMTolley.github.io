This is supposed to be viewed on the web at the URL: "WMTolley.github.io"

However if you wish to download it on github and run it locally then follow the next instructions:
    1. Download python
    2. In a terminal located on "WMTOLLEY.GITHUB.IO" type the command: `python -m http.server`
    3. In a webrowser type into the URL: "http://localhost:8000"
