For 1/8 inche wood: Download Rock Base File then import it in LightBurn. Then set the power and speed of the color red so that it cuts all the way through. Similarly set the power and speed of the color black to however you want to engrave it. Cut out two arcs and however many rectangles you will need (depending on how much space you put inbetween them when gluing). Finally assemble the pieces by gluing the rectangles to both arcs.
